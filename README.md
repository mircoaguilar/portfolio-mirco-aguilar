# Portfolio - Mirco Aguilar

Portfolio personal desarrollado para presentar mi perfil como desarrollador backend, mis proyectos principales y mis enlaces profesionales.

Incluye una breve presentación, stack tecnológico, proyectos destacados, acceso al CV y medios de contacto.

## Sitio en línea

🔗 [Ver portfolio](https://portfolio-mirco-aguilar.vercel.app)

## Secciones

* **Inicio**: presentación personal y resumen profesional.
* **Sobre mí**: perfil orientado al desarrollo backend y enfoque de trabajo.
* **Tecnologías**: stack y herramientas con las que trabajo actualmente.
* **Proyectos**: selección de proyectos desarrollados, con foco en soluciones reales.
* **Contacto**: acceso a LinkedIn, GitHub, email y CV.

## Proyectos destacados

### Sistema de Gestión Comercial

Sistema web desarrollado para un cliente real. Incluye punto de venta (POS), control de stock, gestión de productos y proveedores, historial y reportes de ventas, ranking de productos más vendidos, administración de usuarios con permisos e impresión de tickets.

**Tecnologías:** Node.js, Express.js, PostgreSQL, Bootstrap

### ViajAR

Plataforma web orientada a la gestión integral de servicios turísticos, con reservas de hoteles, transporte y tours, gestión de usuarios, carrito de compras y paneles administrativos.

**Tecnologías:** PHP, JavaScript, HTML, CSS, MySQL
